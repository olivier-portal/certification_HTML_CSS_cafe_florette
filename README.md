# Cafe Florette

A modern, responsive static website for Cafe Florette, a charming coffee shop dedicated to providing exceptional coffee experiences.

## About the Project

This project is a static HTML/CSS website built to showcase Cafe Florette's offerings, location, and contact information. The site features a clean, minimalist design with a focus on user experience and search engine optimization.

## Structure

- **index.html** - Home page featuring the cafe's welcome message and main content
- **where.html** - Location page with information on where to find the cafe
- **contact.html** - Contact page for customer inquiries and feedback
- **css/** - Stylesheet directory containing base styles and resets
- **img/** - Image assets directory

## Features

- Responsive design optimized for all devices
- SEO-friendly meta tags for better search engine indexing
- Clean CSS reset for consistent styling across browsers
- Accessible HTML5 semantic structure
